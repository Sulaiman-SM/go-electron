# go-electron

# pre-requesits
make sure the following are installed
1- Nodejs
2- npm
3- Golang

# to start the project:
git clone git@github.com:Sulaiman-SM/go-electron.git

# start the frontend:
1- cd go-electron/electroni
2- npm insatll
3- npm start

# start the backend:
1- cd go-electron/golang
2- go run server.go
