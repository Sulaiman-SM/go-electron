# go-electron

# pre-requesits
make sure the following are installed
* Nodejs
* npm
* Golang

# to start the project:
`git clone git@github.com:Sulaiman-SM/go-electron.git`

# start the frontend:
* `cd go-electron/electroni`
* `npm insatll`
* `npm start`

# start the backend:
* `cd go-electron/golang`
* `go run server.go`
